# 🛒 ShopFlow App

A simple Android app built using **Kotlin** and **Jetpack Compose** that displays a basic Shop Flow screen.

## ⚙️ Tech Used
- Kotlin
- Jetpack Compose
- Android Studio

## 🚀 How to Run
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or physical device
